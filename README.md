# Portfolio — Adrien Martin

Site portfolio personnel développé avec Flask.

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Le site sera accessible sur http://localhost:5000

## Pages

- `/` — Accueil avec hero, présentation et aperçu des projets
- `/projets` — Tous les projets GitHub avec filtres par technologie
- `/activites` — Centres d'intérêt, domaines de passion et parcours

## Personnalisation

Modifier `app.py` pour :
- Mettre à jour les projets dans la liste `projects`
- Ajouter/modifier les activités dans la liste `activities`
- Mettre à jour les liens LinkedIn et email

## Tech Stack

- **Backend** : Flask (Python)
- **Frontend** : HTML5, CSS3, JS vanilla
- **Fonts** : Playfair Display, Tenor Sans, Lora (Google Fonts)
- **Design** : Brandboard Adrien Martin (Navy #003366, Bordeaux #9E1B32, Crème #F4F1E1)
